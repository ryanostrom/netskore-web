# Ember-Symfony

## Client

* Ember Front-end using pods
* Includes Foundation, Bourbon, Neat and pre-loaded sass settings

#### Installation

* npm install && bower install

#### Deploy

* Serve dist/ for http://example.com

## Server

* Symfony 3
* RESTful API

#### Installation

* composer install

#### Deploy

* Serve web/ for http://api.example.com
